## Receipt Bot 

![Bot Status](https://img.shields.io/endpoint?url=https://receipt-bot-kt3i.onrender.com/health)

A powerful and bot designed to process, analyze, and organize receipts. 

This tool uses **Gemini API** to extract key information from receipts, making expense tracking and financial management effortless.

### Features

- **Smart Receipt Recognition** - Automatically detects and extracts receipt details from photo
- **Data Extraction** - Pulls out items, amounts, dates, and vendor information, saves to the personal data storage
- **Expense Tracking** - Organize and categorize receipts for better financial insights
- **OCR Integration** - Advanced optical character recognition for accurate data capture

### Set up

If you wish to set up this bot for yourself, clon this repo and run setup script:

``` 
bash setup.sh 
``` 

After that, edit contents of `data/bot_config/keys.json`.
